# 高效导航指南

欢迎阅读 ATBCmder 导航指南。本指南涵盖了在文件系统中导航以及管理视图的主要方式。

## 树状视图
树状视图展示了文件系统的层级结构，包括压缩包等虚拟文件系统（VFS）。
它支持三种布局模式，可通过**显示**菜单进行切换：

- **树状视图（替换）**：使用树状视图替换文件列表。

- **树状视图（分栏）**：在当前面板中并排显示树状视图与文件列表。

- **树状视图（两侧面板）**：在左侧和右侧面板中同时启用分栏视图。

右键点击树状视图可以切换**显示文件**选项，以便在树状图中一并显示文件和目录。

## 文件夹标签页与收藏（目录热键列表）
### 文件夹标签页
您可以在左侧和右侧面板中以标签页形式打开多个文件夹。
右键点击标签栏可调用新建标签页、关闭标签页及标签页导航等功能选项。

### 收藏标签组（Favorite Tabs）
您可以将当前整个双面板的标签页布局（包括左右两侧所有打开的标签页）保存为一个具名的“收藏标签组”。

- **保存与加载**：右键点击标签栏，选择**将当前标签页保存为新收藏标签组**，或加载已保存的标签组。

- **管理**：前往**偏好设置 → 收藏标签组**，即可对保存的标签组进行重命名、删除或重新排序。

### 目录热键列表（书签功能）
目录热键列表相当于常用文件夹的书签。

- **快速访问**：使用热键列表弹出菜单快速跳转至书签文件夹。弹出菜单内置搜索框，支持按名称或路径实时过滤。

- **管理**：在**偏好设置 → 目录热键列表**中管理您的书签，支持添加、删除、编辑和重新排序热键条目。

## 快速搜索
快速搜索允许您直接通过输入文件名跳转到对应文件。

- **触发方式**：根据您的设置（**偏好设置 → 快速搜索**），直接输入字母或使用快捷键（如 `Option+字母` 或 `Cmd+Option+字母`）即可调出快速搜索悬浮窗。

- **导航控制**：使用 `Up` 和 `Down` 方向键在匹配的文件之间切换。按 `Enter` 键打开对应文件或文件夹。

- **匹配规则**：您可以配置为匹配文件名开头或匹配名称中任意位置。在输入结尾添加一个点号（`.`）可专门用于精确匹配不含扩展名的文件名末尾。

## 常用快捷键
ATBCmder 高度依赖键盘操作。快捷键按使用场景（如主窗口 Main、文件面板 FilePanel、编辑器 Editor、查找文件 FindFiles）进行了分类管理。
核心快捷键包括：

- `Tab` / `Shift+Tab`：在左右面板或树状视图之间切换焦点。

- `Ctrl+PgDn`：打开压缩包。

- `Ctrl+PgUp`：返回上一级目录。

- `Ctrl+Left` / `Ctrl+Right`：将焦点或内容传输至对侧面板。

您可以为任何命令绑定最多两个快捷键（主快捷键与副快捷键）。如需自定义快捷键或查看各场景的完整快捷键列表，请访问**偏好设置 → 快捷键**。如果您尝试使用已被占用的快捷键，编辑器会自动发出警告。

## 自动调整列宽
默认情况下，文件面板会自动调整各列的宽度，确保您可以完整查看文件名。

- **最大文本宽度（默认）**：列宽自动适配当前可见的最长文件名。

- **平均文本宽度**：根据平均文本宽度加上适当边距来计算列宽。

- **固定宽度**：保持固定宽度；当您手动调整某列大小时会自动激活此模式。

您可以在**偏好设置 → 文件视图**中修改自动调整列宽模式。
