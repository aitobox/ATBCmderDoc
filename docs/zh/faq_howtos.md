# 常见问题与操作指南

本指南针对使用 ATBCmder 过程中的常见问题，提供基于任务的快速参考与解答。

## 通用操作

### 如何在左右面板之间切换？
按下 **`Tab`** 键即可在当前面板与非当前面板之间切换焦点。

### 如何一次性查看某个目录及其所有子目录中的所有文件？
使用**分支视图（平铺视图）**。按下 **`Cmd+B`** 即可激活或关闭该视图。系统将递归展示所有文件的平铺列表。按下 `Esc` 或 `Backspace` 可返回标准目录浏览模式。

### 如何了解某个按钮或选项的功能？
将鼠标悬停在按钮、菜单项或输入框上。ATBCmder 拥有丰富详尽的悬停提示，能够解释功能原理、提供关联的快捷键，并在适用时提供高级说明。

## 文件操作

### 如何复制、移动或删除文件？
ATBCmder 使用经典的 Double Commander 快捷键进行常用文件操作：
- **复制：** 选中文件并按下 **`F5`**。
- **移动/重命名：** 选中文件并按下 **`F6`**。
- **新建文件夹：** 按下 **`F7`**。
- **删除：** 选中文件并按下 **`F8`**。

### 如何在不打开外部软件的情况下预览文件？
选中文件并按下 **`F3`**。ATBCmder 内置了针对音频、PDF 和文本的专属预览器，允许您直接在软件内部预览内容。

### 如何连续播放多个音频文件？
选中多个音频文件（如 `.mp3`、`.flac`）并按 `F3`。内置音频播放器会自动构建播放列表。您可以点击“后台播放”按钮将播放器最小化到右侧面板，在音乐播放的同时继续工作。如需添加更多歌曲，只需在播放器后台运行时选中新歌曲并按 `F3` 即可。

## 高级搜索与命令

### 如何使用自然语言搜索文件？
ATBCmder 提供了**自然语言命令（Semantic Command）**输入功能。
1. 按 **`/`** 或 **`Cmd+F`** 在当前面板底部打开输入框。
2. 输入自然语言指令，如 *"选择所有大于 1MB 的文件"* 或 *"查找今天修改过的 PDF 文件"*。
3. 默认情况下在当前目录进行搜索。若要执行全局 Spotlight 搜索，请在命令前加上前缀 `//`（例如 `//today modified pdf`）。
4. 输入 `/help` 查看更多示例，或输入 `/history` 重放最近使用过的命令。

## 压缩包与网络文件系统

### 如何解压压缩包？
您不必一定要解压它！ATBCmder 将压缩包（`.zip`、`.tar`、`.7z`）视作虚拟文件系统（`vfs://`）。只需选中压缩包并按 `Enter` 键即可进入内部浏览，就如同普通文件夹一样。如果您仍希望提取文件，可以将其从压缩包内复制（`F5`）到对侧面板的标准目录中。

### 如何修改 ZIP 压缩包内部的文件？
导航进入压缩包内部，打开您想编辑的文件并保存更改。ATBCmder 的 RepackWorker 会检测到变更，并自动提示您将文件重新打包回压缩包中。

### 如何连接 FTP、SFTP、WebDAV 或 Samba？
使用**网络虚拟文件系统（Network VFS）**功能。连接建立后，远程文件系统将直接显示在面板中，您可以像对本地目录一样进行复制、移动、重命名和删除等操作。所有连接均采用安全管理，不会将密码保存到配置文件中。

## 故障排查与配置

### 为什么当磁盘上的文件更改时，文件面板没有实时更新？
ATBCmder 具备**自动刷新**功能，可能配置了不同的触发条件。请在“选项”对话框中检查设置：
- 确保已启用自动刷新。
- 检查是否勾选了“当 ATBCmder 处于后台时禁用自动刷新”。
- 确认您当前所在的目录未被列入排除路径列表中。

### 如何在不损坏现有配置的情况下测试新设置？
您可以在独立的测试环境中运行 ATBCmder。使用 `ATBCmder_test.sh` 脚本，它会加载单独的 `.test_config` 配置文件。这允许您安全地试验内部 `cm_*` 命令和快捷键设置。
