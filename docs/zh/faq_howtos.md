# 常见问题与操作指南

本指南提供基于任务的快速参考，并解答关于使用 ATBCmder 的常见问题。

## 基础使用

### 如何在左右面板之间切换？
按下 **`Tab`** 键即可在当前活动面板与非活动面板之间切换焦点。

### 如何一次性查看某个目录及其所有子目录中的文件？
使用 **分支视图（平铺视图）**。按 **`Cmd+B`** 即可激活或关闭该视图，系统将递归平铺展示所有文件。按 `Esc` 或 `Backspace` 可返回标准的目录浏览模式。

### 如何了解某个按钮或选项的功能？
将鼠标悬停在按钮、菜单项或输入框上。ATBCmder 具备极其丰富的悬停提示，能够解释功能作用、显示对应的键盘快捷键，并在适用情况下提供高级帮助。

## 文件操作

### 如何复制、移动或删除文件？
ATBCmder 沿用了经典的 Double Commander 快捷键进行基础文件操作：

- **复制：** 选中文件后按 **`F5`**。

- **移动/重命名：** 选中文件后按 **`F6`**。

- **新建文件夹：** 按 **`F7`**。

- **删除：** 选中文件后按 **`F8`**。

### 如何在不打开外部应用的情况下预览文件？
选中文件并按 **`F3`**。ATBCmder 内置了音频、PDF 和文本的专属预览器，允许您直接在软件内部预览内容。

### 如何连续聆听多个音频文件？
选中多个音频文件（如 `.mp3`、`.flac`）并按 `F3`。内置音频播放器会自动构建播放列表。您可以点击“后台播放”按钮将播放器最小化至右侧面板，在音乐播放的同时继续工作。如需添加更多歌曲，只需在后台播放状态下选中新文件并按 `F3` 即可。

## 高级搜索与命令

### 如何使用自然语言查找文件？
ATBCmder 拥有 **自然语言命令（Semantic Command）** 功能：

1. 按 **`/`** 或 **`Cmd+F`** 在当前面板底部调出输入框。

2. 输入自然语言指令，如 *"选择所有大于 1MB 的文件"* 或 *"查找今天修改过的 PDF 文件"*。

3. 默认情况下在当前目录中搜索。如需进行全局 Spotlight 搜索，请在命令前加上前缀 `//`（例如 `//today modified pdf`）。

4. 输入 `/help` 查看示例，或输入 `/history` 重放最近使用过的命令。

## 压缩包与网络文件系统

### 如何解压压缩包？
您不一定非要解压！ATBCmder 将压缩包（`.zip`、`.tar`、`.7z`）视作虚拟文件系统（`vfs://`）。只需选中压缩包并按 `Enter` 键即可进入内部，像浏览普通文件夹一样操作。如果您仍希望提取文件，可以直接将压缩包内的文件复制（`F5`）至对侧面板的标准目录中。

### 如何修改 ZIP 压缩包内部的文件？
直接进入压缩包，打开需要编辑的文件并保存修改。ATBCmder 的 RepackWorker 会自动识别更改并提示您重新打包更新压缩包。

### 如何连接至 FTP、SFTP、WebDAV 或 Samba？
使用 **网络虚拟文件系统（Network VFS）** 功能。连接成功后，远程文件系统将显示在面板中，您可以像操作本地目录一样进行复制、移动、重命名和删除等交互。所有连接均安全管理，不会将密码明文保存至配置文件中。

## 故障排查与配置

### 为什么当磁盘文件发生改变时面板没有自动更新？
ATBCmder 拥有 **自动刷新** 功能，其配置可能有所不同。请检查“设置（选项）”对话框中的配置：

- 确认已开启自动刷新。

- 检查是否设置了当 ATBCmder 处于后台时禁用自动刷新。

- 确认您当前所在的目录没有被加入排除路径列表。

### 如何在不破坏现有配置的情况下测试新设置？
您可以在独立的测试环境中运行 ATBCmder。使用 `ATBCmder_test.sh` 脚本加载单独的 `.test_config` 配置文件，方便您安全地试验内部 `cm_*` 命令与快捷键。
